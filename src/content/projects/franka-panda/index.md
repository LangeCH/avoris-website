# Überblick

Im Rahmen eines industriellen Projekts entwickelte ich eine automatisierte Fertigungszelle auf Basis eines Franka Panda Roboters.

# Technologien

- Python
- OPC UA
- Node-RED
- Franka RIDE
- Pneumatik

# Aufgaben

- Entwicklung der Roboterlogik
- Kommunikation über OPC UA
- Erstellung von HMIs mit Node-RED
- Integration pneumatischer Komponenten
- Inbetriebnahme und Fehlersuche

# Herausforderungen

Die größte Herausforderung war die zuverlässige Kommunikation zwischen den verschiedenen Komponenten der Anlage sowie die Abstimmung von Roboter-, HMI- und SPS-Prozessen.

# Ergebnis

Es entstand ein funktionsfähiges Automatisierungssystem mit grafischer Bedienoberfläche und automatisierter Prozesssteuerung.
